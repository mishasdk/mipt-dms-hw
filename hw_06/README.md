# hw_06

Проект с ДЗ_6 "Автоматизация расчетов с помощью Apache AirFlow".

## Запуск DAG

1. Развернуть сервисы с помощью `docker-compose.yaml`:

```sh
docker compose up -d
```

2. Зайти в airflow UI клиент по адресу [0.0.0.0:8080](https://0.0.0.0:8080).

3. Вручную создать новый connection к базе данных, с id `postgres`. user, password и db вписать airflow.

4. Далее, перейти на вкладку DAG и запустить флоу.

5. Уведомление на email в случае невыполненной проверки можно проверить через mailhog UI клиент [0.0.0.0:8025](https://0.0.0.0:8025).
